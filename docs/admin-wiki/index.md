# Admin Wiki

A short, friendly guide for everyone who manages the website's content. No coding
required.

There are **two ways** to manage content:

- **The web admin** — a dashboard you open in a browser, like WordPress.
- **The Telegram bot** — chat with a bot on your phone to post listings on the go.

Both do the same thing behind the scenes: they save your changes and the public
website updates itself a minute or two later.
